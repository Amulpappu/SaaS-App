## 🔑 Generate Required Secrets

```bash
# Run these commands and save the output
openssl rand -hex 32   # SECRET_KEY
openssl rand -hex 16   # POSTGRES_PASSWORD
```

Add to `.env.production`:
- SECRET_KEY
- POSTGRES_PASSWORD  
- STRIPE_SECRET_KEY (from dashboard)
- STRIPE_WEBHOOK_SECRET (from dashboard)
- FRONTEND_URL (your domain)

## 🆘 Troubleshooting Quick Links

**Services won't start?**
→ `docker compose logs` | See: DEPLOYMENT_GUIDE.md Troubleshooting

**Database won't connect?**
→ `docker compose exec backend nc -zv db 5432` | See: DEPLOYMENT_GUIDE.md

**Out of disk?**
→ `docker system df` then `docker system prune -a` | See: DEPLOYMENT_GUIDE.md

**Need to restore backup?**
→ `docker compose exec -T db psql < backup.sql` | See: scripts/backup-db.sh

**Performance issues?**
→ `./scripts/health-check.sh` then review `DEPLOYMENT_CHECKLIST.md`
