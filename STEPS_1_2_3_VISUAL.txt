STEP 1️⃣  - YOUR COMPUTER (Terminal)
════════════════════════════════════════════════════════════════

📍 WHERE:
   Mac:   Click Spotlight → type "terminal" → Press Enter
   Win:   Right-click anywhere → Select PowerShell
   Linux: Open any terminal

📝 WHAT:
   Type this command (then press Enter):
   
   openssl rand -hex 32
   
   Copy the output string!
   
   Type this command (then press Enter):
   
   openssl rand -hex 16
   
   Copy the output string!

💾 SAVE:
   Paste both strings into a text file
   (Example: secrets.txt)


STEP 2️⃣  - YOUR WEB BROWSER (Online)
════════════════════════════════════════════════════════════════

📍 WHERE:
   Open your web browser (Chrome, Firefox, Safari, Edge)
   Go to: https://dashboard.stripe.com

🔐 LOGIN:
   Use your Stripe account (create one if needed)

🔍 NAVIGATE:
   Left menu → "Developers"
   Click → "API keys"

📋 FIND & COPY:
   Look for "Live" section (NOT "Test")
   
   Copy these 2 items:
   1. Secret Key (starts with sk_live_)
   2. Webhook Secret (starts with whsec_)

💾 SAVE:
   Paste both into your text file with Step 1 values


STEP 3️⃣  - YOUR COMPUTER (Terminal Again)
════════════════════════════════════════════════════════════════

📍 WHERE:
   Open the same terminal from Step 1

📝 WHAT:
   
   Command 1:
   cd /path/to/saas_platform
   (Replace with your actual folder path)
   
   Command 2:
   git add .
   
   Command 3:
   git commit -m "Ready for deployment"
   
   Command 4:
   git push origin main

✅ RESULT:
   Your code appears on GitHub
